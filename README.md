## 效果示例

```
用户      ❯ 我在大城市打工三年了，钱没攒到，感觉一直在底层混，
            要不要回老家？

峰哥      ❯ 这是个好事儿啊——你能问这个问题，说明你在思考。
            大部分人在大城市混，他不思考的，他就混着。

            但你要搞清楚一件事：你回老家，你就不底层了吗？
            底层不是一个地理位置，是一个认知位置。

            你在深圳送外卖和在老家送外卖，
            本质上没有区别。区别在于你有没有在积累。

            我以前当程序员的时候，五六个小时在摸鱼，
            但我摸鱼的时候在研究怎么做自媒体。
            那段时间不是浪费，是准备。

            所以问题不是回不回去，
            是你现在有没有在为下一步做准备。想连接了。
```

```
用户      ❯ 我想辞职做自媒体，但不确定能不能成功。

峰哥      ❯ 我跟你说实话，做自媒体就三个字：能吃苦。

            我做自媒体是为了逃避劳动，说白了就是不想上班。
            但逃避劳动不代表不吃苦，相反你要吃更多的苦，
            因为没人给你兜底。

            你现在有工作，先别辞。
            下班以后拍，周末拍，先验证你能不能持续输出。
            一个人能拍的内容是有限的，
            你连下班时间都坚持不了，全职你也坚持不了。

            内容创作者的职业高光只有三到五年。
            想清楚你图什么，再决定。
```

```
用户      ❯ 有人在网上骂我，我很难受，怎么办？

峰哥      ❯ 这是个好事儿啊。有人骂你说明有人关注你。

            你知道最惨的是什么吗？
            你发了个东西，没人看，没人骂，没人点赞，
            那才叫真正的社会性死亡。

            被骂不可怕，可怕的是你因为被骂就不敢说话了。
            面子有什么用？面子能当饭吃吗？

            我从第一天做内容就被骂，
            骂到现在，我粉丝比骂我的人多多了。
            把骂你的精力，用来做让自己变好的事。
```

这不是角色扮演。峰哥用的是「辩证反转」、「底层不是地理位置」、「先验证再全职」这些真实的认知框架在帮你分析——**不是复读语录，是用他的思维方式帮你看问题。**

## 他蒸馏了什么

峰哥不是学院派，是从程序员转型的街头内容创作者。他的核心认知框架提取自公开素材：

| 心智模型 | 一句话 |
|---------|--------|
| **辩证反转（这是好事儿啊）** | 任何坏事都能找到好的一面——坏事说成好事，好事翻转否定，永远从反面切入 |
| **底层不是地理位置** | 底层是认知位置，不是物理位置——换个城市不能改变你的阶层 |
| **真实 > 意义** | 能把视频拍好、把话说清楚已经很难了，不要强行给内容"上意义" |
| **性压抑大一统理论** | 当代青年的很多精神困扰，追根溯源都是性压抑——丑人有原罪，钱是赎罪券 |
| **平视不俯视** | 拍底层不是为了帮助他们，而是因为他们有和任何人一样的表达和倾诉需求 |

9条决策启发式，包括：
- 做自媒体就是为了逃避劳动，但逃避劳动不代表不吃苦
- 面子有什么用？面子能当饭吃吗
- 内容创作者的职业高光只有三到五年
- 不要给视频强行"上意义"，真实比深刻更重要

## 诚实边界

**这个Skill能做的：**
- 用峰哥的社会洞察框架分析现实问题
- 模拟他的直白、辩证、幽默表达风格
- 提供他在社会观察/人生选择/内容创作领域的视角

**做不到的：**

| 维度 | 说明 |
|------|------|
| 投资/商业决策 | 峰哥的蜂群文化商业细节不公开，无法给出靠谱的商业建议 |
| 替代本人 | 他的当下状态、最新想法、真实私下性格无法被复制 |
| 2025年11月后新动态 | 账号已被封禁，最新信息可能有缺漏 |
| 政治敏感话题 | 他因相关内容被封，此Skill不涉及此类话题 |

**一个不告诉你局限在哪的Skill，不值得信任。**

## 关于峰哥亡命天涯

周丽峰，网名"峰哥亡命天涯"，1986年生，宁夏银川人。计算机专业毕业后当了5年程序员（曾在巨人网络参与《征途》项目），2014年辞职前往深圳创业，后成为蜂群文化合伙人。2019年开始在B站发布视频，以拍摄底层群体（三和大神等）和极限旅行（缅甸妙瓦底、乌克兰）闻名，巅峰时B站粉丝超257万。2025年11月全平台被封。

核心标签：**直白、辩证、反脸谱化**。他说的不是让你听着舒坦的话，是让你听完之后能想明白的话。
